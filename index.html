About This System

This system allows users to fill placeholders in text prompts effortlessly. Here's how it works:

1. **Enter Prompt**: Users can enter a text prompt containing placeholders (e.g., [name], {date}).

2. **Generate Placeholders**: By clicking the "Generate Placeholders" button, the system identifies placeholders in the prompt and generates corresponding input fields.

3. **Fill Placeholders**: Users fill in the generated input fields with the desired values.

4. **Preview**: As users fill in the placeholders, the system updates a preview of the prompt with the filled values in real-time.

5. **Copy to Clipboard**: Users can copy the filled prompt to the clipboard by clicking the "Copy to Clipboard" button.

6. **Sample Prompts**: Users can click the "Sample Prompts" button to view and use predefined sample prompts.
